# frontend — Interface web

SPA qui consomme les API via la gateway : inscription/login, gestion des organisations et documents, chat RAG, flux d'audit live.

**Module :** Framework frontend (React) — Web — *Majeur (2 pts, partagé)*

## À faire
- Auth (login, 2FA, OAuth), gestion organisations & membres.
- Dépôt / consultation de documents selon le rôle.
- Chat avec l'assistant RAG (réponses + citations).
- Affichage du flux d'audit temps réel (WebSockets).

## Standards
- ESLint (`@typescript-eslint`) + Prettier, TypeScript `strict`.
- Composants fonctionnels + hooks.

**Stack :** React + Vite + TypeScript, Tailwind CSS
